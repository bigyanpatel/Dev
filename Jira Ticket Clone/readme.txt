This is a basic Jira Ticket Clone app made with basic HTML, CSS & JavaScript.

Live Website: https://jira-ticket-bigyan.netlify.app/

This uses CRUD operations to
  - create tickets with specific color id tags.
  - it can read all the tickets and filter them based on their respective IDs.
  - updates all the tickects after locking or unlocking the edit option.
  - selected tickets can be deleted from the storage directly.

Browser storage API is used to avoid loss of data after every browsing sessions.
